# Barcode Generator  :earth_americas:


## Install  :gear:

```console
1. Clone this repo
    $ git clone https://github.com/d4niel9/barcode_generator.git

2. Change the working directory
    $ cd barcode_generator

3. Install the requirements
    $ pip install -r requirements.txt

5. Run the script
    $ python3 main.py
```


## Use :video_game:
